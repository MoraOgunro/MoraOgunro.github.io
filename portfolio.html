Music Player 2.0 (Java)

A pair of Android applications which utilize custom API integration and proxy communication. When used together, function as a music player similar to Spotify or Soundcloud. The server application creates a service and a custom AIDL API. The client calls the API through a binded proxy and can play songs and retrieve information about the song directly from the server. This app follows all of the current Android standards for custom API’s, AIDL, and custom services. This is a comprehensive upgrade to my original Music Player app.

Threaded Tic-Tac-Toe AI (JavaFX)

A game of Tic-Tac-Toe utilizing AI, concurrent programming, thread safety, and a full JavaFX GUI. Each opponent runs on it’s own synchronized thread using Android handlers, loopers, and messages. Each thread also contains its own AI, which it uses to intelligently determine the next move. The UI thread is left unhindered and responsive as a result of diverting computation to the worker threads.When the game is over, the UI thread dynamically signals the concurrently running threads to stop.

IMDB App (Java)

An Android application that displays a TV show’s IMDB page in a WebView at the press of a button. The UI is constructed out of fragments that respond intelligently to the orientation of the screen. The app also communicates with two other Android applications using broadcast receivers, ordered intents, and custom permissions.

.NET MySQL Superhero Satabase w/ GUI

A functional GUI application to query a database of Superhero information using C#, MySQL, and .NET connector libraries. The user can run queries on this database by selecting the corresponding button and the results will be displayed within the GUI.

Music Player 1.0 (Java)

An Android application that utilizes the full power of the YouTube API, allowing the user to play the song, view clips, or go to the artist’s Wikipedia page. The UI makes use of Recycler Views, context menus, and View adapters.
